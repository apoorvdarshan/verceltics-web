# Verceltics Website

Landing page for [Verceltics](https://github.com/apoorvdarshan/verceltics) — a mobile analytics viewer for Vercel.

**Live**: [verceltics.site](https://verceltics.site)

## Pages

- `index.html` — Landing page
- `privacy.html` — Privacy Policy
- `terms.html` — Terms of Service

## Deploy

Static site. Deploy to Vercel:

```bash
vercel --prod
```
